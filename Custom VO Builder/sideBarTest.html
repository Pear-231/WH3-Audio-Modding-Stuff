README

This sheet is for custom battle VO.

Column A contains drop downs of all the dialogue events you can choose from.

To use this builder first you select a dialogue event from the drop down. This will populate a set of state groups.

Next select State Path from the drop down. This will populate all the possible states you can select within each state group. Each State Path must be unique.

Once you've chosen your states, select Sounds from the drop down. For this put the paths to your sounds - one per cell - for example "D:\Ole_is_a_donkey.wav".

Next either add another state path or select another dialogue event and repeat the process.
