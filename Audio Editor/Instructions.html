VO Builders

NOTE: Before building any VO, go to the menu and click Audio Editor then Load States, and Load Dialogue Events.

Column A contains drop downs of all the Dialogue Events you can choose from for the type of VO you are building, as well as a an option called "State Path".

To build your VO, first select a Dialogue Event from the drop down. This will automatically populate the Dialogue Event row with the State Groups for that Dialogue Event, and create rows for "State Path" and "Sounds". The "State Path" row contains drop downs for all the possible States within each State Group.

Then create your State Path by selecting a State from the drop down. Each State Path should be unique.

Next, on the Sounds row, add the file paths to the sounds you want the State Path to play (include as many sounds as you like). Each cell should contain only one file path e.g.

Cell B4:
"D:\Ole_is_a_donkey.wav"

Cell B5:
"D:\Dank_not_based_memes.wav"

Cell B6:
"D:\Ace_is_a_womaniser.wav"

Cell B7:
"D:\Kou_runs_warhammer_hubðŸ‘€.wav"

Cell B8:
"D:\grackul_likes_hammers.wav"

Finally either add another State Path or select another Dialogue Event and repeat the process.

Once you're finished building your VO, go to the menu and click Audio Editor then Download Sheet as Audio Project.

Modded States

This sheet contains your custom modded states.

Add new States to the relevant State Group column then go to the menu and click Audio Editor then Load States.

The names of your States should ideally follow vanilla naming conventions.
